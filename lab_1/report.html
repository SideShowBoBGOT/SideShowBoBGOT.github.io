Лабораторна 1

Панченко Сергій ІП-11

7 Варіант

Мета:
Ознайомлення з мовою розмітки гіпертексту HTML та базовими засобами CSS для візуалізації веб-сторінок.
Висновок:
Під час лабораторної роботи зверстав сайт відповідно до свого варіанту: використовував ul-, li-, frame-, a-теги; локальна файлова система працює та дозволяє заходити на інші сторінки сайту через посилання. Розмістив сайт на githubpages. Для розміщення елементів використовував Flex CSS. У наступній роботі продемонструю роботу з використанням лише Grid Css.
